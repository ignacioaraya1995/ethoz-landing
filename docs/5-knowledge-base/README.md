## Como usar esta base de conocimiento

1. **Antes de escribir cualquier pieza de contenido**, consultar al menos dos archivos relevantes.
2. **Para datos y cifras**, siempre citar desde `estadisticas.md`. Nunca inventar numeros.
3. **Para diferenciacion**, consultar `competencia.md` y `posicionamiento-ethoz.md` juntos.
4. **Para urgencia legal**, usar `leyes-regulaciones.md` con fechas exactas.
5. **Para storytelling**, basarse en `historias-casos-uso.md`. Adaptar, no copiar literal.
6. **Para tono y frases**, revisar `frases-citas.md` antes de escribir el headline.
7. **Para segmentar mensajes**, consultar `audiencias-messaging.md` segun el canal y publico.

## Principios editoriales de Ethoz

- **Tono**: Simple, seguro, profesional. Sin jerga tecnica innecesaria. Sin exageracion.
- **Voz**: Directa, humana, emponderada. Hablamos como aliados, no como vendedores.
- **Datos**: Siempre con contexto. Un numero sin historia no mueve a nadie.
- **Urgencia**: Real, no fabricada. Las leyes tienen fechas. Los riesgos son concretos.
- **Respeto**: Nunca culpar a los educadores. El sistema los fallo, no al reves.
